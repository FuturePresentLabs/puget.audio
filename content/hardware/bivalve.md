# BiValve

**Stereo Multi-Filter**

Seven legendary filter circuits, modeled circuit-accurate, in stereo. Cycle through Moog ladder, MS-20, SSM2044, Polivoks, Steiner-Parker, Wasp, and Jupiter-6 — all on one button.

## Filter Library

1. **Moog Ladder** — LP / HP-tap
2. **MS-20** — LP / HP
3. **SSM2044** — LP / HP-tap
4. **Polivoks** — LP / HP-tap
5. **Steiner-Parker** — LP / HP
6. **Wasp** — LP / HP
7. **Jupiter-6** — LP / HP / BP / Notch

## Key Features

- **Skew knob** — Offset cutoff between L/R channels for true stereo movement
- **Stereo processing** — Dual filters, independent or linked
- **Circuit-accurate models** — WDF-based, component-level precision
- **CV control** — Cutoff, resonance, and mode selection

## Specifications

- **Format:** Eurorack, 3U, 10HP
- **Audio:** 24-bit / 96kHz
- **I/O:** Audio in/out, CV in (1V/oct)
- **Power:** +12V: 150mA, -12V: 50mA
- **Dealer:** $199 (MAP $279)
