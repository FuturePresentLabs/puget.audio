# SuperPedal

**Hardware Multi-FX**

A hardware stompbox running our full WDF engine. SuperPedal brings Puget Audio's analog simulations to your pedalboard with true bypass, expression pedal input, and USB-C for firmware updates and patch management.

## Features

- **True bypass** — No tone suck when off
- **Expression pedal input** — Real-time parameter control
- **USB-C** — Firmware updates and patch management
- **Includes all Puget Legends algorithms** — GOLDENROD, RATKING, SCREAMER, SUPER
- **Future FX support** — New algorithms via firmware updates

## Specifications

- **Processor:** Daisy Seed — ARM Cortex-M7 @ 480MHz
- **Audio:** 24-bit / 96kHz
- **Latency:** < 1ms
- **Power:** 9V DC, 300mA (standard pedal supply)
- **Connectivity:** USB-C, Expression pedal
- **Enclosure:** Die-cast aluminum
