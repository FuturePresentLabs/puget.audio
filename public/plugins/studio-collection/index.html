Why WDF matters for a full signal chain

Each plugin in the collection uses the same PedalKernel WDF engine. They share the same mathematical rigor and the same approach to circuit modeling. When you chain them together, you get the same kind of interaction you’d get chaining the real hardware — because each plugin is modeling real components with real impedance characteristics.

The 1073’s output transformer drives the 1176’s input transformer. The LA-2A’s tube stages color the signal before the 670’s push-pull amplifier. These interactions aren’t programmed — they emerge from the physics.
