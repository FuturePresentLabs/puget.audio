DrummerBoy

4-Voice Drum Module

A 4-voice drum module in Eurorack format. Each voice is a complete WDF circuit model — pull from a library of classic machines: 808 and 909 kicks, snares, and hats, Simmons SDS-V, LinnDrum, Oberheim DMX, plus a Reese bass voice.

Voice Library

Kicks:

  • 808 Kick (bridged-T)
  • 909 Kick
  • LinnDrum Kick
  • Oberheim DMX Kick

Snares:

  • 808 / 909 Snare

Hats:

  • 808 Hats
  • 606 Hats

Other:

  • 808 Cowbell / Metallic
  • Clap
  • Sub Bass (V/oct)
  • Simmons SDS-V Tom
  • Reese Bass

Specifications

  • Format: Eurorack, 3U, 16HP
  • Voices: 4 independent, mixable
  • Processor: Daisy Seed · M7 @ 480MHz
  • Display: 128×64 OLED
  • Audio: 24-bit / 48kHz
  • Power: +12V 200mA · −12V 80mA
  • Dealer: $199 (MAP $249)

I/O

  • TRIG — Trigger / Gate per voice
  • ACCENT — Per-voice accent CV
  • OUT — Voice out (norm. to mix)
  • MIX OUT — Stereo mix w/ level
  • GATE — Global gate input
  • V/OCT — Global pitch input
